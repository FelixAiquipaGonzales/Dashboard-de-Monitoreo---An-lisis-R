# Contributing

Gracias por interesarte en contribuir a este proyecto. Para mantener la calidad y facilitar la colaboración, sigue por favor estas indicaciones:

- Abre un issue antes de trabajar en cambios importantes para discutir el enfoque.
- Crea ramas con nombres descriptivos: `feature/descripcion` o `fix/descripcion`.
- Escribe commits claros y atómicos. Usa mensajes en español o inglés de forma consistente.
- Abre Pull Requests con una descripción del cambio, pruebas (si aplica) y capturas/ejemplos.
- Asegúrate de que el dashboard se renderiza con `rmarkdown::render()` y que no expones datos sensibles.

Si necesitas ayuda con el entorno o dependencias, abre un issue y te orientamos.
