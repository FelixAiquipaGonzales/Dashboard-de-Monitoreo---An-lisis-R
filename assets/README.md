# Assets

Coloca aquí las imágenes y recursos que aparecerán en el `README.md` y en el dashboard.

- `preview.png` — Imagen de portada mostrada en el README.

Para agregar la imagen de portada, copia el archivo `preview.png` en esta carpeta y luego realiza un commit:

```bash
git add assets/preview.png
git commit -m "Add project preview image"
```
